================
Introducción
================

Sistema ETL completo para la extracción incremental de datos desde PostgreSQL
a MinIO, con limpieza automática de datos.

**Características:**

- ✅ Extracción incremental desde PostgreSQL
- ✅ Almacenamiento en capas (Bronce → Silver)
- ✅ Limpieza automática de datos
- ✅ Gestión de estado persistente
- ✅ Detección automática de columnas de seguimiento

Ver la documentación técnica completa en :doc:`documentacion`.
